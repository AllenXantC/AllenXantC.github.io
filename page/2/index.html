关系

发表于 2024-10-05 更新于 2024-10-27 分类于 哲学 ， 数理逻辑

XantC

关系的定义只关乎两点：受限制的对象与限制条件。对象的部分依靠卡氏积给出，限制条件就是限制条件。

卡氏积：

\[ X\times Y=\lbrace(x,y)\mid x\in X \wedge y\in Y\rbrace \]

卡氏积本身只是给出了两个元素的有序性，这也正是关系所需要的。一个真正的关系是：

\[ R\subseteq X\times Y \]

关系一般会再加上一些限制条件，如\(R=\lbrace (m,n)\in \mathbb{Z}\times\mathbb{Z}\colon m\mid n\rbrace\)，代表整除关系。

关于势能

发表于 2024-10-04 分类于 物理

XantC

势能这个概念，从它自身来看非常奇怪。为什么一定会有这种东西？为什么它叫做“势”？这个中文字本身不太好解，我决定用英文来说。势=potential，势能就是拥有转化为其他能量的潜力的能。也就是说，它是一个需要通过其他能量来反推出来的概念。

比如说引力势能。一个物体从地球周围向中心加速，它的动能自然会增加。而根据能量守恒，这增加的能量一定有一个来源，要么是来自它身上其他形式的能，要么是其他物体上的能。而在场的只有物体和地球，地球的能量显然几乎没有变化，因而物体本身一定存在另一种形式的能，用于转化为动能。

在我的想法里，引力势能就是这样被发现，并拥有了这个名字的。所以说，没有动能或其他能的变化，人类不会想到去回溯地规定一个势能出来。这是一种辩证法。
